## Objetivo

Este exercício contém uma função de envio de imagens vulnerável. Não realiza qualquer validação dos ficheiros que os utilizadores enviam antes de os armazenar no sistema de ficheiros do servidor.

Para resolver o exercício, envie um web shell básico em PHP e utilize-o para extrair o conteúdo do ficheiro /home/carlos/secret. Envie este segredo utilizando o botão disponibilizado no banner do exercício.

Pode iniciar sessão na sua própria conta utilizando as seguintes credenciais: wiener:peter
